Installation

Quick Installation Guide
  • Go to Settings > Plugins.
  • Install the Telegram-Notification-for-RISE-CRM.zip which you have downloaded from the Envato Market.
    It'll require the item purchase code. Don't use any nulled version for your security.
  • If everything is OK, it'll show a success message.
  • Activate the plugin from the option dropdown.

Settings

You'll get the Telegram Integration settings in Settings > Plugins > Telegram Notification > Telegram integration.

You need Bot Token and Chat ID to integrate. Please follow the steps below to get them:

  1. Open search panel and search @BotFather.
  2. Type /newbot and reply like the below image.

    You'll get your Bot Token on this step. Add this to the integration setting.
  3. Now go to your desired group where you want to get notification. Then go to group settings. Click on add member button.
  4. Search for the bot you've created earlier (RiseCrmBot) and add that.
  5. Again go to your group settings and click on add member button. Then search @getidsbot and add that.
  6. After adding this, you'll get your Chat ID.
  7. Add this credentials to the integration settings and click on Save.
  8. After saving the credentials you'll get a button to test the notification. Click on that to check if it's working properly.
  9. Now go to Telegram notification settings tab and enable notifications for your desired events.
    Please note that there must be web notification enabled in app notification settings to get Telegram notification..

Upgrading Guide

Please note that you can only download the updates if you had a valid purchase code on installation.

Please backup all files and database before start the installation of updates.

There is a very easy way to update your application.

To install any updates go to Settings > Plugins > Telegram Notification > Dropdown > Updates. The download will be start automatically once any new update is available. Once the download is completed you can install the updates by clicking on the link. That’s all!
